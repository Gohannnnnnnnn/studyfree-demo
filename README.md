# 医邦教育

这是一个适合小范围使用的在线课程学习平台。它现在采用更接近学习通这类学习软件的课程空间结构：老师围绕一门课程发布任务点、公告、签到、讨论、测验和作业，学生在同一个课程页面完成学习和互动。

## 医邦教育 Windows 免费版定位

医邦教育 是一个独立实现的免费课程学习平台，目标是覆盖课程、资料、作业、测验、签到、讨论、公告和学习进度这些核心教学流程。

它不会破解会员，不会绕过登录或付费限制，不会复制学习通代码、品牌、素材，也不会使用学习通私有接口。学习通只作为功能类别参考，本项目使用自己的代码、界面和本地数据。

## Windows 桌面运行

开发电脑已安装 Node.js 20 或更新版本时，可以双击：

```text
run_windows.bat
```

脚本会安装依赖并启动 医邦教育 桌面窗口。数据保存在当前 Windows 用户的应用数据目录中，上传资料也保存在本机。

## Windows 打包

在开发电脑上双击：

```text
build_windows.bat
```

脚本会先安装依赖、运行测试，然后生成可分发目录：

```text
release\医邦教育-win32-x64\医邦教育.exe
```

把整个 `release\医邦教育-win32-x64` 文件夹复制到其他 Windows 电脑即可运行。

## 已实现功能

- 学生端、教师端、管理员端登录入口。
- 学生注册、教师注册、邀请码限制。
- 学生年级固定选择一年级、二年级、三年级，课程适用年级也使用同一组选项。
- 教师注册后需要管理员审核。
- 教师创建课程、发布/下架课程、删除自己的课程。
- 教师上传章节资料，作为课程任务点。
- 学生打开课程资料并标记学习进度。
- 课程公告：教师发布，学生查看。
- 课堂签到：教师发起签到码，学生输入签到码完成签到。
- 讨论区：学生提问，教师或学生回复。
- 随堂测验：教师发布简答题，学生提交后自动按标准答案判分。
- 作业：教师发布作业，学生提交内容。
- 学习统计：教师查看学生学习进度。
- 后台管理：用户审核、邀请码管理、查看所有课程、修改课程基础信息、发布/下架或删除课程。

## 使用流程

1. 管理员登录后台，审核教师账号。
2. 教师登录教师端，创建并发布课程。
3. 教师进入课程空间，添加任务点、公告、签到、讨论、测验和作业。
4. 学生注册并登录学生端，进入课程空间学习。
5. 学生完成任务点、签到、测验、讨论和作业。
6. 教师在学习统计和课程空间中查看学生参与情况。

## 云服务器部署

安装 Docker 后，在项目目录运行：

```bash
docker compose up -d --build
```

默认端口是 `3000`。正式部署时建议：

- 在服务器安全组开放 80/443，并使用 Nginx 反向代理到 3000。
- 设置环境变量 `TOKEN_SECRET` 为随机长字符串。
- 定期备份 `data/` 和 `uploads/`。
- 后期人数增多时，将 JSON 存储迁移到 PostgreSQL，将上传文件迁移到对象存储。

## 数据目录

- `data/db.json`：用户、邀请码、课程、章节、进度、公告、签到、讨论、测验、作业等数据。
- `uploads/`：教师上传的课程资料。
